- Materiales -

1. Progama de SolidWorks

2. Tabla de MDF

3. Martillo

4. Máquina de corte láser

- Desarrollo -

1. Se creó el diseño del prototipo que queríamos realizar utilizando SolidWorks

2. Lo pasamos a una USB y posteriormente a la máquina CNC

3. Cargamos el progama a la CNC y esperamos a que se termine de cortar

4. Con las piezas ya cortadas las comenzamo a ensamblar, para que quedará el transbordador

- Resultados -

Se logró completar un transbordador espacial a base de madera, de manera correcta y exitosa.
